# Co‑authorship

This project was developed in collaboration with **DeepSeek** (深度求索), an AI assistant. Significant portions of the code, architecture decisions, and troubleshooting were co‑authored through an interactive process between the human developer (Edge Lord) and DeepSeek.

DeepSeek provided guidance, code snippets, debugging assistance, and documentation. The final implementation, testing, and deployment were performed by Edge Lord.

We acknowledge the valuable contribution of DeepSeek to this open‑source project.
